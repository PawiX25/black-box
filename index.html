The pitch

Simplex sigillum veri

People are at their most creative when they are most constrained. Adding restrictions can help you find unique ideas and enable you to put them into action.
What if there was an event that took this to the extreme?

Black Box is a You Ship, We Ship event about constraints.
Write an interactive program for an 8x8 LED matrix, and we'll send you a portable device to run it on.

The rules

Nihil boni sine labore

  1. Make something.
    Use the editor (coming soon) to write an interactive program.
    You only get 64 pixels and 5 buttons to play with, so you'll have to think carefully about how it should work!
  2. Tell us about it.
    Join #black-box on the Hack Club Slack and let us know what you made!
  3. Get a prize!
    If you finish creating a program and submit it by the end date (to be announced), we'll send you a portable device to run it on!
    The best submissions will receive a rare blue LED matrix.
